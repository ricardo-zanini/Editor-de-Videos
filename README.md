# Editor de Vídeos - Fundamentos de Processamento de Imagens

### Projetado em Java utilizando a biblioteca "Swing"

- Inverte Imagens
- Aplicação de Filtro Cinza
- Ajuste de Brilho
- Ajuste de Contraste
- Negativo de imagem
- Rotações
- Filtros de pixels
- ...
